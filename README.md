# invistica-web

Frontend da plataforma Invística. Next.js 16 + TypeScript + Tailwind CSS.

## Setup local

```bash
npm install
npm run dev
```

Acesse http://localhost:3000

## Documentos de referência

- `CLAUDE.md` — contexto técnico para assistentes IA
- `PRODUCT.md` — posicionamento estratégico e roadmap
- `docs/` — mockups e referências visuais

Next.js 16 · React 19 · TypeScript · Tailwind CSS · shadcn/ui · Framer Motion · Inter
